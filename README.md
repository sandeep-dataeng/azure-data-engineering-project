# Azure Data Engineering Project

## Overview
This project demonstrates an end-to-end data pipeline using Azure Data Factory, PySpark, and Azure Synapse.

## Tools Used
- Azure Data Factory
- Azure Synapse Analytics
- PySpark

## Workflow
1. Data ingestion from CSV files
2. Metadata-driven processing
3. Data transformation
4. Data loading into sink

## Features
- Dynamic pipelines using parameters
- Scalable architecture
- Automated ETL workflows
